    .c-button

    Since v2.0.0
    .c-button { ... }

    Description

    Button component styles. Contains Primary and Secondary styles.

    The "--noStyle" is primarily used for modal 'X' or similar where it should have the accessible functionality of a button, but removes the base styles. These styles can be used on links or buttons.

    Use a <button> if it is performing an action, such as submitting a form. Use a <a> if it is directing the user to another page.

    • CTA's in hero images often go to another page. Therefore they should be links, <a>.
    • A Modal close icon often closes the modal. Therefore it should be a <button>.

    Author

    • Kevin Kendle
